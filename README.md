# AI Note Summarizer

An AI-powered note summarizer built using HTML, CSS, JavaScript, Node.js, Express, and OpenRouter API.

## Features

* AI note summarization
* Upload text files
* Download summary
* Dark mode
* Summary history
* Copy summary

## Technologies

* HTML
* CSS
* JavaScript
* Node.js
* Express
* OpenRouter API

## Run

npm install

node server.js

Open index.html with Live Server
