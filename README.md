# 🧠 AI Note Summarizer Pro

An AI-powered note summarizer built using HTML, CSS, JavaScript, Node.js, Express, and OpenRouter API.

This project allows users to paste notes or upload text files and generate concise AI summaries through an API connection.

---

## ✨ Features

* AI-powered note summarization
* Upload `.txt` files
* Download summary as `.txt`
* Copy summary button
* Dark mode
* Character counter
* Word counter
* Summary history
* Clear notes
* Responsive interface

---

## 🚀 Installation

Clone repository:

git clone https://github.com/anjanaradhakrishnan-tech/ai-note-summarizer.git
Open project:

cd ai-note-summarizer

Install dependencies:

npm install

Create `.env`

API_KEY=YOUR_API_KEY

Start backend:

node server.js

Open:

index.html

using Live Server.

---

## 📌 How It Works

User enters notes
↓
Frontend sends request
↓
Backend receives request
↓
OpenRouter processes summary
↓
Summary returned to website

---

## 🔒 Security

API keys are stored in `.env`.

`.gitignore` prevents sensitive files from being uploaded.
